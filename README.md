This project is a Single Page Application (SPA) infographic visualizing a strategic framework for restructuring K-12 education in Nigeria through Artificial Intelligence and digital transformation.

Overview

The infographic covers:

Current Landscape: Challenges regarding digital access and teacher availability.

Proposed Structure: A new 3-tier educational system (Foundational, Skills, Specialization).

AI Integration: How AI tutors and administrative automation can bridge the gap.

Global Benchmarks: Comparisons with Finland, South Korea, and the USA.

Projected Impact: Forecasts on STEM literacy and job readiness.

Technologies Used

HTML5: Semantic structure.

Tailwind CSS: Responsive styling and layout.

Chart.js: Data visualization and interactive charts.
